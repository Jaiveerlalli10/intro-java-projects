## Todo List Manager

A console-based task management app written in Java. Users can add, remove, 
load, and save tasks — with an optional extension to auto-sort tasks by length.

## Features
- Add tasks at any position in the list
- Mark tasks as done (removes them)
- Load a task list from a file
- Save the current list to a file
- Extension: sort tasks by character length

## How to Run
1. Compile: `javac TodoListManager.java`
2. Run: `java TodoListManager`

## Concepts Used
- ArrayLists and dynamic data structures
- File I/O with Scanner and PrintStream
- Input validation
- Sorting algorithm (selection sort by string length)
