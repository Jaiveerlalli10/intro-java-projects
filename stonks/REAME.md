# Stonks — Stock Trading Simulator

A console-based stock trading simulator written in Java. Users load a stock 
dataset from a file, then buy and sell shares, view their portfolio, and save 
their holdings to a file.

## Features
- Load stock data from a formatted text file
- Buy shares with a minimum $5 budget
- Sell shares (only if you own enough)
- Display current portfolio holdings
- Save portfolio to a file for later

## How to Run
1. Compile: `javac Stonks.java`
2. Run: `java Stonks`
3. When prompted, enter a stock data file (e.g. `stocks.txt`)

## Concepts Used
- Arrays and parallel arrays
- File I/O with Scanner and PrintStream
- Modular method design
- Input validation and error handling
